# Metrika Project Configuration

## Browser Settings
Default browser: Safari

When using browser tools or launching the application, always use Safari as the default browser.

## Project Information
This is a React + TypeScript + Vite project for the Metrika application.

### Tech Stack
- React 18.x
- TypeScript 5.x  
- Vite 5.x
- Zustand (state management)

### Development
- Run `npm run dev` to start the development server
- The application runs on http://localhost:5173
